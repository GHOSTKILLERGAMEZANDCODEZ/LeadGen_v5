## [5.1.0] — 2026-03-20

### ✨ Новое

#### GUI

##### Генератор ссылок Яндекс.Карт
- **Новая компоновка интерфейса**: сегмент+результаты слева, города справа
- **Часовые пояса городов**: встроенная функция определения времени (`utils/city_timezones.py`)
- **Диалог управления городами и районами**: модальное окно для редактирования списка городов
- **Умное распределение**: выбранные города сохраняются между сессиями

##### HomePage
- **Логотип приложения**: добавлен визуальный элемент на главную страницу
- **Улучшенная типографика**: обновлённые шрифты и отступы
- **Glassmorphism эффекты**: карточки с полупрозрачным фоном

#### Утилиты
- **city_timezones.py**: функция `get_city_timezone(city)` для определения часового пояса
- **Улучшенный config_loader**: поддержка вложенных конфигураций

### 🔧 Изменения

#### Рефакторинг GUI
- **Генератор ссылок**: полная переработка компоновки (`link_generator_page.py`)
  - Использование `QSplitter` для разделения левой/правой панели
  - Перемещены города из верхней части в правую панель
  - Добавлена прокрутка для списка городов
  - Улучшена валидация ввода сегмента

- **Диалог управления городами**: выделен в отдельный компонент
  - `QDialog` для редактирования городов и районов
  - Поддержка добавления/удаления городов
  - Сохранение в `config.json`

#### Настройки
- **Удалено управление городами из настроек**: перемещено в диалог генератора ссылок
- **Удалено дублирование кнопок "Добавить город"**: единая точка управления

#### Стили
- **Glassmorphism theme**: добавлены переменные для стеклянных эффектов
  - `GLASS_CARD_BG`
  - `GLASS_BORDER`
  - `GLASS_BUTTON_BG`
  - `GLASS_HOVER`

### 📦 Зависимости

#### Обновления
- Без изменений

### 🧪 Тесты

- TODO: тесты для `city_timezones.py`

## [5.0.0] — 2026-03-20

### ✨ Новое

#### Архитектура
- Добавлен **Dependency Injection Container** (`core/dependency_container.py`) для централизованного управления зависимостями
- Добавлен **ProcessingService** (`modules/services/processing_service.py`) — сервисный слой для абстракции между GUI и бизнес-логикой
- Создана **иерархия исключений** (`modules/exceptions.py`) с 10 специализированными типами ошибок:
  - `LeadGenError` (базовое)
  - `SecurityError`
  - `BitrixWebhookError` и подтипы (`NotFoundError`, `ForbiddenError`, `RateLimitError`, `InvalidURLError`, `ConnectionError`)
  - `ProcessingError`
  - `ValidationError`
  - `ExportError`

#### GUI
- Полная реорганизация структуры GUI:
  - Перемещены страницы в `gui/pages/` (новая модульная структура)
  - Перемещены компоненты в `gui/components/`
  - Перемещены стили в `gui/styles/`
  - Перемещены потоки в `gui/threads/`
  - Перемещены утилиты в `gui/utils/`
- Добавлен **круговой прогресс-бар** (`circular_progress.py`)
- Добавлен компонент **FileLoader** с поддержкой Drag & Drop
- Добавлен компонент **FileList** для отображения списка файлов
- Улучшена навигация через Sidebar с анимацией

#### Документация
- Полностью переработан **README.md** с добавлением:
  - Архитектурной диаграммы слоёв
  - Документации API BitrixWebhookClient
  - Схемы базы данных
  - Примеров использования всех модулей
- Обновлён **QWEN.md** с информацией о:
  - ProcessingService и DependencyContainer
  - Иерархии исключений
  - Новой структуре GUI
- Создан **CHANGELOG.md**

### 🔧 Изменения

#### Миграция на PySide6
- Полная миграция с PyQt6 на **PySide6** (официальный Qt для Python)
- Обновлены все импорты: `PyQt6` → `PySide6`
- Обновлены требования в `requirements.txt`: `PySide6>=6.5.0`

#### Рефакторинг
- Выделен сервисный слой для уменьшения связанности GUI и бизнес-логики
- Улучшена обработка ошибок через специализированные исключения
- Централизовано создание зависимостей через DI Container

#### Удаление мёртвого кода
- Удалены устаревшие файлы:
  - `gui/dark_theme.py` (заменён на `gui/styles/theme.py`)
  - `gui/analytics_tab.py` (заменён на `gui/pages/analytics_page.py`)
  - `gui/link_generator.py` (заменён на `gui/pages/link_generator_page.py`)
  - `gui/lpr_finder.py` (заменён на `gui/pages/lpr_finder_page.py`)
  - `gui/manager_monitor_tab.py` (заменён на `gui/pages/manager_monitor_page.py`)
  - `gui/settings_tab.py` (заменён на `gui/pages/settings_page.py`)
- **Итого удалено:** ~1600 строк мёртвого кода
